## Fitur Aplikasi
1.  **Create:** Menambah data barang baru.
2.  **Read:** Melihat stok barang di tabel.
3.  **Update:** Mengubah data stok dan harga.
4.  **Delete:** Menghapus data barang.

## Teknologi yang Digunakan
*   **Bahasa:** Java (JDK 17)
*   **Framework:** Spring Boot
*   **Database:** MySQL
*   **GUI:** Java Swing

## Cara Menjalankan
1.  Import database `proyek_rpl2_inventaris` di phpMyAdmin.
2.  Buka project di NetBeans.
3.  Run file `AplikasiInventaris.java`.
